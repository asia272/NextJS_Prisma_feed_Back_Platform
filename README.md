# 🚀 Feedback Fusion

**Feedback Fusion** is a modern full-stack web application that allows users to submit, manage, and analyze feedback efficiently. It features secure authentication, role-based access, and a scalable architecture for real-world use.

---


## 📌 Features

- 🔐 Secure authentication with Clerk  
- 📝 Submit and manage feedback  
- 👥 Role-based user system (candidate / interviewer)  
- ⚡ Fast and responsive UI  
- 📊 Structured feedback management  
- 🌍 Scalable full-stack architecture  

---

## 🛠️ Tech Stack

**Frontend**
- Next.js  
- React

**Backend**
- Next.js (Full-stack / API Routes)
